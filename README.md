# AI Agent Series

Building an AI agent from scratch, one stage at a time -- from a bare
echo loop up to an agent with tools and evals. Each stage lives in its own
folder under `agents/`, paired with a blog post in `docs/` explaining it.

- **Code**: [`agents/`](agents/) -- one self-contained folder per stage.
- **Posts**: [`docs/`](docs/) -- published via GitHub Pages.

See [CLAUDE.md](CLAUDE.md) for repo conventions and commands.
